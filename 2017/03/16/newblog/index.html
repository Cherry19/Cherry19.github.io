实验一 在Github上用Markdown创建技术博客

一、实验目的
1.熟悉Github网站
2.掌握Markdown语法
二、实验任务
1.注册Github账号
2.创建Gihub Pages repository
3.发布第一篇博客
4.学习Markdown
5.利用Hexo生成博客并部署到Github
三、实验步骤与结果

  1. 注册GitHub账号并且创建 repository

2.本地配置Hexo
1)安装git
2)安装node
3)设置npm镜像为淘宝
npm config set registry https://registry.nom.taobao.org
4)安装hexo
npm install -g hexo-cli
5)初始化站点
hexo init myhexo
cd myhexo
npm install
6)浏览本地站点
hexo server
3.在GitHub上部署hexo
1)在GitHub上新建repository，名称必须是用户名.github.io，比如我的是Cherry19.github.io
2)修改配置文件_config.yml的deploy部分和site部分，注意空格

3)安装hexo-deployer-git
npm install hexo-deployer-git –save
4）配置git全局邮箱与名字
git config –global user.email “2984487900@qq.com”
git config –global user.name “Cherry19”
5）生成博客
hexo generate
6）部署博客
hexo deploy
4.添加新文章
1）生成新文章
hexo new newblog
2)用编辑器修改文章：myhexo\source_posts\newblog.md
3)生成并发布到GitHub
hexo d -g
实验结果：
https://cherry19.github.io/

四、实验小结
1.一定要注意当第二次打开cmd 的时候一定要执行cd myhexo。
2.在GitHub上新建repository，名称必须是用户名.github.io，比如我的是Cherry19.github.io
3.修改配置文件_config.yml的deploy部分和site部分，注意空格
4.还有邮箱用户名设置一定要填自己注册博客时候的邮箱和博客名
5.细心一点，耐心一些。
